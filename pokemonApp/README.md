### A Pokemon game:

This is nice implementation of `__pyORM__` in a simple Pokemon game, the game is based on a public API (`https://pokeapi.co/api/v2/`)

__Requirements__:
* python +3.6 is required

__Libraries__:

- requests
- tqdm
- PyInquirer
- prompt-toolkit==1.0.14
- prettytable
- pyfiglet

### Setup and run

```bash
$ pip3 install -r requirements.txt
$ python3 main.py
```
